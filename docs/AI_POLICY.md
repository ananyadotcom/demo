# AI Permitted Use Policy

Version: 1.0

Policy Owner: Repository Administrator

Approval Date: 25-Jun-2026

Review Date: 25-Jun-2027

## Purpose

This policy governs the acceptable use of artificial intelligence tools, including GitHub Copilot, ChatGPT, and similar AI-assisted development technologies.

## Permitted Uses

* Code suggestions and autocompletion
* Documentation generation
* Unit test generation
* Refactoring assistance
* Code review assistance

## Prohibited Uses

* Uploading confidential information to unauthorized AI services
* Bypassing security reviews
* Automatically merging AI-generated code without human review
* Using AI-generated content without required disclosure

## Human Review Requirement

All AI-generated code must undergo human review and approval before merge.

## Exceptions

Any deviation from this policy requires documented approval and entry into the exception register.
