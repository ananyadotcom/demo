# Prompt and Output Retention Procedure

Document Owner: Repository Administrator

Version: 1.0

Approval Date: 26-Jun-2026

Next Review Date: 26-Jun-2027

## Purpose

This procedure defines how AI prompts and AI-generated outputs are retained for governance, audit, and compliance.

## Scope

Applies to all contributors using AI tools such as GitHub Copilot and ChatGPT.

## Retention Requirements

- AI prompts must be documented when required by project policy.
- AI-generated outputs must be retained in the approved repository or external retention system.
- Evidence may be stored in pull requests, issues, or the approved retention platform.

## Review

Compliance is reviewed quarterly by the Repository Administrator.
